REPORT OF WEBSITE

Introduction

This project focuses on creating a structured and responsive website using HTML and CSS. The design ensures compatibility across both desktop and mobile devices. The website consists of multiple pages, including a Home Page, Project Page, Contact Page, Video Demo Page, and a Site Report Page. The styling is managed using separate CSS files for desktop and mobile, ensuring an optimized user experience. Website Structure The website is designed with different sections, each having a specific function are navigation Bar: Contains links to different pages for easy access and Main Content Area: Displays relevant content based on the selected page and Footer: Includes copyright information. The HTML structure follows a well-organized format, with external CSS files applied for styling.

HTML Implementation

The website consists of multiple HTML files, each designed for different sections. Key features include are a responsive design, ensuring accessibility on both mobile and desktop and a structured navigation menu linking all pages and a Semantic HTML tags to improve readability and accessibility. The Site Report Page is designed to showcase the overall project, highlighting the coding structure and styling methods.

CSS Styling

The website utilizes two separate CSS files are reportdesktop.css – Used for styling on larger screens (above 800px width) and reportmobile.css – Applied for screens smaller than 800px. Key CSS Features are body Styling: Background color, margin settings, and font styles and navigation Menu: Inline list items styled with hover effects for smooth transition and Header & Footer: Fixed positioning for better user experience.

Conclusion

This project successfully demonstrates the creation of a responsive, well-structured website using HTML and CSS. The combination of separate styling for mobile and desktop improves usability. The navigation, layout, and structure contribute to an efficient and user-friendly interface.
